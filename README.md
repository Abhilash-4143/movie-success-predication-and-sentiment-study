# 📌 Overview

**CineMetrics AI** is a production-inspired Data Science web application that combines **Machine Learning**, **Natural Language Processing**, and **Interactive Analytics** to predict movie success and analyze audience sentiment.

The platform demonstrates an end-to-end data science workflow—from preprocessing movie datasets to training predictive models, performing NLP-based sentiment analysis, and visualizing insights through an interactive dashboard.

# 📦 Installation

Clone the repository

```bash

```

Go into the project

```bash

```

Install dependencies

```bash

```

Run development server

